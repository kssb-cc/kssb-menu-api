KSSB Menu API

Welcome to this thing.
It is really, really basic.

Available Endpoints

  • /api/get_raw: Returns a dictionary of the entire menu. Each key represents an indevidual day. Note that you cannot yet get lunch, for example, unless you split by I.E. "Lunch: \n". This may get fixed at some point...
  • /api/get: If you supply a day, it will return the string of that day. If the day is not provided or there is no menu for that day, it will return "Wrong day".
